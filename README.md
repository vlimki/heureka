# heureka - back up ideas quickly via git

```
heureka 0.0.1 - back up ideas via git
Usage: heureka [help|setup|new]

help - Prints this help menu
setup - Sets up a git repository
new [filename] - opens up an editor with a new note. Will generate a filename if called with no `filename` argument.

examples:
- `heureka new idea1`
- `heureka new`
```
